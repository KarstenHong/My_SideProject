# 📝 智能筆記整合系統

## 功能特色

### ✨ 自動美化排版
系統會自動辨識您貼入的文字格式，並套用專業的視覺樣式：

#### 支援的格式標記

1. **標題格式**
   - 全大寫文字自動轉為標題
   - 數字編號（1. 2. 3.）自動格式化
   - 中文編號（一、二、三）自動識別

2. **文字強調**
   - `**粗體文字**` → **粗體文字** (紫色加粗)
   - `*斜體文字*` → *斜體文字* (紫色斜體)
   - `_底線文字_` → <u>底線文字</u>
   - `【重點標記】` → <mark>高亮顯示</mark>

3. **列表格式**
   - `• 項目` 或 `- 項目` → 帶左側紫色邊框的項目
   - `1. 項目` → 自動數字編號格式

4. **特殊區塊**
   - `> 引用文字` → 淺藍背景引用框
   - `` `代碼` `` → 灰色背景代碼標記
   - `---` 或 `===` → 虛線分隔線

5. **自動辨識**
   - 網址自動轉為可點擊連結
   - 段落自動增加行距
   - 適當的留白和間距

### 📊 管理功能

- **分類管理**：可自訂類別，自動分類整理
- **標籤系統**：為筆記加上多個標籤
- **搜尋過濾**：快速找到需要的筆記
- **即時預覽**：查看美化後的排版效果
- **編輯更新**：修改後自動重新排版

### 📄 匯出功能

- **PDF 匯出**：保留所有格式的專業文件
- **Word 匯出**：可編輯的 .doc 格式
- **分類整合**：自動按類別分組輸出

## 使用範例

### 輸入文字：
```
會議主題

本次討論事項：
- 專案進度檢討
- 預算調整方案  
- 下季度規劃

**重點決議**
1. 專案延期兩週
2. 追加預算 10%
3. 增聘兩名工程師

> 主管意見：需要加強團隊溝通

參考資料：https://example.com
```

### 自動美化後：
系統會將文字轉換為：
- 「會議主題」→ 紫色大標題
- 項目列表 → 左側紫色邊框
- **重點決議** → 紫色粗體
- 數字列表 → 紫色編號
- 引用 → 淺藍背景框
- 網址 → 可點擊的連結

## 快速開始

1. 開啟 `NoteRecord.html`
2. 選擇或新增類別
3. 輸入標題
4. 貼上您的文字內容（系統會自動美化）
5. 加上標籤（選填）
6. 點擊「新增筆記」
7. 使用「預覽排版」查看效果
8. 匯出為 PDF 或 Word 文件
