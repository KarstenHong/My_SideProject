開啟 `home.html` 選擇您需要的功能：

- **📝 快速點餐** → 適合新增訂單

## 📝 快速點餐流程

### 開啟頁面

```
開啟 order.html
```

### 操作步驟

1. **選擇菜品**（左側）
   - 點擊 `+` 增加數量
   - 點擊 `−` 減少數量
   - 數字會即時更新

2. **填寫資訊**（右側）
   - 訂單號碼 \*
   - 姓名 \*
   - 聯絡電話 \*
   - 所屬群組 \*
   - 備註（選填）

3. **確認送出**
   - 檢查右下角的總金額
   - 點擊「✅ 送出訂單」
   - 確認成功後會自動清空表單

### 💡 小技巧

- ✨ 直接在左側網格快速選菜
- 🔄 誤操作可點「重置」按鈕
- 📱 手機也能輕鬆使用（響應式設計）

---

## 📊 訂單管理流程

### 開啟頁面

```
開啟 management.html
```

### 主要功能

#### 1. 查看統計

- 📈 總訂單數
- 💰 總金額
- 🏆 熱門菜品

#### 2. 管理菜品

```
點擊「🍽️ 菜品管理」→「➕ 新增菜品」或「📋 管理現有菜品」
```

- 新增新菜品
- 編輯菜品價格
- 刪除不需要的菜品

#### 3. 查詢訂單

```
使用搜尋列或群組篩選
```

- 輸入姓名、電話、訂單號碼
- 選擇特定群組
- 點擊「搜尋」

#### 4. 查看訂單詳情

```
點擊訂單列的「查看」按鈕
```

- 顯示完整訂單資訊
- 列出所有訂購菜品

#### 5. 編輯訂單（需管理員權限）

```
按 Ctrl+Shift+A 啟用管理員模式
輸入密碼：admin2026
```

- 點擊「編輯」修改訂單
- 點擊「刪除」移除訂單

#### 6. 匯出報表

```
點擊頂部的匯出按鈕
```

- 📊 匯出 Excel：完整資料表格
- 📄 匯出 PDF：列印友善格式

#### 7. 匯入訂單

```
點擊「📥 匯入 Excel」
選擇 Excel 檔案
```

- 自動檢查重複訂單
- 提供覆蓋或跳過選項

## 🔐 管理員模式

### 啟用方式

**方法1：鍵盤快捷鍵**

```
按 Ctrl + Shift + A
輸入密碼：admin2026
```

**方法2：Console 命令**

```javascript
// 在瀏覽器開發者工具的 Console 中執行
toggleAdminMode("admin2026");
```

### 啟用後可以

- ✏️ 編輯現有訂單
- 🗑️ 刪除訂單
- 🔍 查看完整個資（不隱碼）

### 關閉方式

```
再按一次 Ctrl + Shift + A
或執行：toggleAdminMode()
```

## 🆘 常見問題

### Q: 為什麼我看不到編輯/刪除按鈕？

**A:** 需要先啟用管理員模式（Ctrl+Shift+A）

### Q: 訂單資料會不會遺失？

**A:** 不會，所有資料都即時同步到 Firebase 雲端

### Q: 可以同時在兩個頁面操作嗎？

**A:** 可以，資料會即時同步

### Q: 如何修改管理員密碼？

**A:** 在 `ChineseNewYear_Dishes.js` 中修改 `correctPassword` 變數

### Q: 手機可以使用嗎？

**A:** 可以！所有頁面都有響應式設計

### Q: 舊版頁面還能用嗎？

**A:** 可以，`ChineseNewYear_Dishes.html` 仍保留完整功能
