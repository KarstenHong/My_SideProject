# 洪師傅2026金馬年菜訂購系統 - 拆分版本說明

## 📋 系統架構

系統已成功拆分為兩個獨立頁面，並保留所有原有功能：

### 1️⃣ **快速點餐頁面** (`order.html`)

- ✅ **一頁式設計** - 無需滾動即可完成所有操作
- 📱 左右分欄佈局：
  - **左側**：菜品選擇區（網格卡片式設計）
  - **右側**：客戶資訊填寫與訂單總計
- 🎯 特色功能：
  - 快速增減數量按鈕（+ / −）
  - 即時顯示訂單總金額
  - 一鍵重置所有內容
  - 直觀的視覺回饋效果

### 2️⃣ **訂單管理頁面** (`management.html`)

- 📊 完整的管理功能：
  - 統計面板（總訂單數、總金額、熱門菜品）
  - 菜品管理（新增、編輯、刪除菜品）
  - 訂單列表（查詢、編輯、刪除訂單）
  - 分頁顯示
  - 搜尋與篩選
  - 匯出功能（Excel、PDF）
  - 匯入功能（Excel）

### 3️⃣ **導航首頁** (`home.html`)

- 🏠 系統入口頁面
- 清楚展示兩個主要功能
- 列出系統特色
- 提供舊版介面連結

### 4️⃣ **舊版完整頁面** (`ChineseNewYear_Dishes.html` 或 `index.html`)

- 保留原始的單頁完整版本
- 包含所有功能於一頁

---

## 🚀 使用方式

### 開始使用

1. 開啟 **`home.html`** 作為系統入口
2. 選擇您需要的功能：
   - **點餐** → 前往 `order.html`
   - **管理** → 前往 `management.html`

### 點餐流程

1. 進入 **order.html**
2. 在左側選擇菜品（點擊 + 增加數量）
3. 在右側填寫訂購資訊
4. 確認總金額後點擊「✅ 送出訂單」

### 管理流程

1. 進入 **management.html**
2. 查看統計數據
3. 管理菜品（新增、編輯、刪除）
4. 查詢訂單（搜尋、篩選、編輯、刪除）
5. 匯出報表（Excel、PDF）

## 🔧 技術特點

### 一頁式點餐的實現

```css
/* 使用 Flexbox 實現左右分欄 */
.order-content {
  display: flex;
  height: calc(100vh - header高度);
  overflow: hidden;
}

.dishes-panel {
  flex: 1.5; /* 菜品區佔較大空間 */
  overflow-y: auto; /* 可滾動 */
}

.customer-panel {
  flex: 1; /* 資訊區佔較小空間 */
  display: flex;
  flex-direction: column;
}
```

### 菜品網格佈局

- 使用 CSS Grid 自動排列
- 響應式設計，自動調整列數
- 卡片式設計，直觀易用

### 數據同步

- 兩個頁面共用相同的 JavaScript 檔案
- Firebase 即時同步資料
- 訂單資料在兩頁面間保持一致

## 🔐 權限管理

### 管理員模式

在任一頁面按下 **Ctrl + Shift + A** 啟用管理員模式：

- 預設密碼：`admin2026`
- 啟用後可編輯和刪除訂單
- 關閉後自動隱藏敏感功能

### 個資保護

- 非管理員模式自動隱碼：
  - 姓名：顯示姓氏，名字用 ○ 代替
  - 電話：保留前4碼和後3碼
  - 群組：顯示前兩字
